# Theme Playground

A simple website to preview UI components across different visual themes.

This app lets you switch between multiple themes and instantly see how the UI looks, including colors, fonts, spacing, and components.

---

## Available Themes

- Claude  
- Neobrutalism  
- Supabase  
- Vercel  
- Mono  
- Notebook  

---

## What you can do

- Switch themes in real time
- Toggle light and dark mode
- Preview common UI components (buttons, inputs, cards, etc.)
- See how each theme behaves visually before using it
